class ml.enums.ExecutionMode(*values)

Bases: Enum

Defines the parallelism mode for an Execution strategy. - THREAD: Parallel parts are executed in separate threads within the same process. - PROCESS: Parallel parts are executed in separate, persistent worker processes.
